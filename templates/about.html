About

This website is a book store that allows users to search for and buy books. It is written in Python using the Flask framework. The front end is written in HTML and uses tailwindcss for styling. The website is a college project written by Rae Noble, the sole developer of the project.

The website is hosted on Heroku and the code is available on GitHub here. The website will use the Stripe API for payment processing.

The website is called Shelfworm, a play on the word bookworm. The website is designed to be simple and easy to use. The website is designed to be responsive and work on all devices. The website is designed to be accessible and easy to use for all users.

The website is a work in progress and is not yet fully functional. The website is a college project and is not intended for commercial use. The website is a prototype and is not intended to be used for real transactions. Perhaps after a rewrite in a more suitable language and framework, like Django, React or Svelte, the website could be used for real transactions.
